# HtyBox v1.5.0

## 2026-06-29

- **双端远程中继（relay）**：手机不在同一 WiFi（异地 / 蜂窝）也能经自托管中继远程查看并操控桌面的终端与工作区。设置 → 连接 新增 relay 开关、中继地址、TLS 与在线状态。
- **修复：连接手机后桌面终端被"压缩"**：连接 iOS 客户端后，桌面终端不再被压成手机尺寸、手机端可正常滚动查看；桌面拖动窗口改尺寸时手机端跟随、不再被截断。
- **全局文件搜索（双击 Shift）增强**：搜索结果可含文件夹；单击文件按设置「直接在编辑器打开」或「仅在文件树定位选中」（方便找文件喂给 AI），单击文件夹在文件树定位；自动展开侧栏并切到 File 页签。
