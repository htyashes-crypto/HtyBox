- 修复了终端里输入中文或中英混排时字符错位、个别字消失的问题，现在输入和反复增删都能正常显示，不必再拖动窗口才恢复
- 优化了终端中文字体显示，更平滑、更贴近系统原生终端的观感
- 新增了在文件页签直接预览图片：支持 PNG / JPG / GIF / WebP / BMP / ICO / AVIF 等格式，棋盘格背景居中显示并标注像素尺寸，不再只提示无法编辑
- 支持了把整个文件夹拖入文件树复制，自动保留内部目录结构（含空文件夹），不再需要逐个拖文件
- 新增了编辑器文件自动同步：打开的文件被外部（如终端里的 Claude / Codex）改动后即时刷新，Markdown / SVG 预览同步更新；本地有未保存改动时会提示可重载或忽略
- 重做了安装程序界面：换用 HtyBox 品牌图标、欢迎页插画与页眉，并改为简体中文，告别默认安装器外观
- 优化了安装界面图像在高分屏上的清晰度
- 优化了文件树拖拽时的上下滚动提示条，从纯色渐变改为细腻颗粒质感，带柔和淡入与轻微流动，外观更精致
- 修复了全局搜索：忽略名单中的文件夹名会误伤到深层的同名文件夹，导致文件树里看得到的文件却搜不到
- 新增了全局搜索的「索引文件数上限」设置（默认 10 万，可调），并在设置与搜索框显示当前工作区文件数及是否超限
- 新增了「更新设置」：可选择启动时自动检查新版本、是否自动下载更新，并能在设置里随时手动检查更新
